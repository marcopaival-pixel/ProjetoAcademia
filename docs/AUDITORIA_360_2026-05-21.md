**Conclusão técnica:** arquitetura Laravel sólida com legado `php-app/` isolado em produção. CI cobre PHPUnit + PHPStan com suíte Feature **verde** (maio/2026). Prioridade seguinte: aumentar cobertura (módulos sem teste), reduzir baseline PHPStan gradualmente e validar smoke em produção.

**Enviar:** `app/`, `bootstrap/`, `config/`, `public/` (+ `build/`), `resources/views/`, `routes/`, `artisan`, `composer.lock`, `vendor/` (ou `composer install --no-dev` no servidor).

## 17. Passo a passo de deploy

Ver **`docs/DEPLOY_NEXSHAPE.md`**. Resumo:

1. Build assets (`npm run build`) e `composer install --no-dev`
2. Upload código + `public/build/`
3. `.env` produção + `php artisan key:generate` (primeira vez)
4. `php artisan migrate --force`
5. Caches `config:cache`, `route:cache`, `view:cache`
6. Supervisor queue + scheduler
7. Smoke: `/up`, `/api/v1/health`, login, webhook MP sandbox
