Solicitação para implementação de ambiente de demonstração com dados simulados

Criar um ambiente de demonstração (Demo) no sistema que utilize dados simulados e permita navegação completa sem permitir alteração de dados reais.

O objetivo é permitir que novos usuários conheçam o sistema antes de contratar um plano.

---

TIPOS DE DEMO

Criar contas de demonstração para:

* Demo Aluno
* Demo Profissional
* Demo Clínica
* Demo Administrador

Cada perfil deve possuir dados simulados específicos.

---

DADOS SIMULADOS

Criar dados fictícios para:

Clínica:

* Nome
* Plano
* Profissionais
* Pacientes
* Créditos de IA
* Financeiro
* Assinatura

Pacientes:

* Nome
* Peso
* Altura
* IMC
* Treino
* Dieta
* Histórico

Financeiro:

* Faturas
* Pagamentos
* Status
* Próxima cobrança
* Créditos de IA

IA:

* Análises
* Recomendações
* Histórico

---

REGRAS DO MODO DEMONSTRAÇÃO

Permitir:

* Navegação
* Visualização
* Consulta de dados
* Uso das funcionalidades em modo simulado

Bloquear:

* Criar registros
* Editar registros
* Excluir registros
* Pagamentos
* Compra de créditos
* Alteração de configurações
* Consumo real de IA

---

SEGURANÇA

Garantir que:

* Nenhum dado real seja alterado
* Nenhuma cobrança seja executada
* Nenhum crédito de IA real seja consumido
* Nenhum webhook seja disparado
* Nenhuma configuração do sistema seja alterada

---

RESET AUTOMÁTICO

Os dados simulados devem ser restaurados automaticamente:

* ao sair do sistema
  ou
* a cada 24 horas

---

IDENTIFICAÇÃO VISUAL

Mostrar banner fixo:

Modo Demonstração

Mensagem:

"Você está utilizando dados simulados. Nenhuma alteração será salva."

---

COMPATIBILIDADE

A demo deve permitir visualizar:

* Pacientes
* Treinos
* Dietas
* Avaliações
* IA
* Financeiro
* Assinaturas
* Relatórios
* Dashboards
* Painel do aluno
* Painel do profissional
* Painel da clínica
* Painel administrativo

---

OBJETIVO FINAL

Criar um ambiente seguro, profissional e realista para apresentação e demonstração do sistema, facilitando vendas e captação de novos clientes.
