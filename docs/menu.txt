Crie no sistema Laravel uma funcionalidade de personalização de menus baseada em assinatura do usuário.

OBJETIVO:

Permitir que usuários que possuem uma assinatura ativa possam configurar quais menus desejam exibir no sistema, personalizando sua interface conforme suas necessidades.

Somente usuários com assinatura ativa poderão acessar essa funcionalidade.

CONCEITO:

Cada usuário poderá:

* Escolher quais menus deseja visualizar
* Ocultar menus que não utiliza
* Salvar sua configuração
* Restaurar o padrão do sistema

Se o usuário não tiver assinatura:

* Os menus padrão do sistema devem ser exibidos
* A opção de personalização deve ficar bloqueada

FUNCIONALIDADES:

1. Verificar se o usuário possui assinatura ativa
2. Exibir lista de menus disponíveis
3. Permitir marcar ou desmarcar menus
4. Salvar preferências do usuário
5. Carregar menus conforme configuração salva
6. Restaurar configuração padrão
7. Permitir controle por assinatura

REGRAS DE NEGÓCIO:

Se:

subscription_status = active

Permitir:

Configurar menus

Se:

subscription_status != active

Bloquear acesso

Mostrar mensagem:

"Esta funcionalidade está disponível apenas para usuários com assinatura ativa."

FUNÇÃO DO SISTEMA:

Quando carregar o sistema:

Buscar menus permitidos para o usuário

Exemplo:

$menus = Menu::whereHas('preferences', function ($query) {
$query->where('user_id', auth()->id())
->where('visible', true);
})->orderBy('order')->get();

Se o usuário não tiver configuração salva:

Carregar menus padrão
