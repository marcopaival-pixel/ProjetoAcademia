1. Criar um novo administrador:

php artisan app:user create admin@exemplo.com --name="Nome do Admin" --password="senha123" --admin --premium

2. Resetar a senha de um utilizador existente:

php artisan app:user reset mestre@academia.com --password="nova_senha_456"

3. Transformar um utilizador comum em Admin/Premium:

php artisan app:user reset email@usuario.com --admin --premium


===============================================================================
Parâmetros disponíveis:

action: Pode ser create ou reset.
email: O e-mail do utilizador (obrigatório).
--name: Nome do utilizador (opcional para create).
--password: A senha desejada (se não informar, o comando irá perguntar).
--admin: Define o utilizador como administrador.
--premium: Define o utilizador como premium (vitalício).
