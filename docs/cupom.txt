Crie no sistema Laravel uma funcionalidade de geração de cupons de desconto para profissionais que possuem assinatura ativa. Todas a setapas que faz é o administrador e quando
o profissional solicitar tem que ter a liberação do administrador.

OBJETIVO:

Permitir que profissionais com assinatura ativa possam gerar cupons de desconto para seus pacientes, sendo que cada cupom deve ser válido apenas para uma pessoa e utilizado apenas uma vez.

REGRAS DE NEGÓCIO:

1. Somente profissionais com assinatura ativa podem gerar cupons
2. Cada cupom deve possuir um código único
3. Cada cupom deve ser utilizado apenas uma vez
4. Cada cupom deve ser válido para apenas um usuário
5. O sistema deve validar a data de validade do cupom
6. O sistema deve registrar o uso do cupom
7. O sistema deve impedir reutilização do cupom
8. O sistema deve permitir cancelar um cupom
9. O sistema deve permitir definir desconto percentual ou valor fixo
10. O sistema deve registrar qual profissional criou o cupom

FUNCIONALIDADES:

* Criar cupom
* Listar cupons
* Enviar cupom para paciente
* Aplicar desconto
* Marcar cupom como utilizado
* Cancelar cupom
* Ver histórico de uso

REGRAS DE VALIDAÇÃO:

Antes de aplicar o cupom, verificar:

Se o cupom existe
Se o cupom está ativo
Se a data não expirou
Se o cupom ainda não foi utilizado
Se o usuário ainda não utilizou o cupom

Incrementar used_count

SEGURANÇA:

Implementar middleware:

CheckActiveSubscription

Para garantir que apenas profissionais com assinatura ativa possam gerar cupons.

OBJETIVO FINAL:

Criar uma funcionalidade segura e profissional de cupons de desconto individuais para pacientes dentro do sistema de academia.
